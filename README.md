# NumPy CLI Analyzer

A command-line based student data analyzer built using **NumPy**.  
This project demonstrates matrix operations and data analysis.

---

## Features

- Add student marks
- Display student data
- Calculate:
  - Student-wise averages
  - Subject-wise averages
- Finds top performer
- Ranks students
- Clears all data
- Saves & load data using JSON

---

## Tech Stack

- Python
- NumPy
- JSON
